Titanic Dataset – Advanced Data Cleaning & EDA
This repository contains a complete workflow for advanced data cleaning and exploratory data analysis (EDA) on a Titanic dataset created in Excel. The project is aimed at identifying patterns and trends influencing passenger survival using Python's data science libraries.

Project Objective
To clean and preprocess raw Titanic data
To engineer new features such as FamilySize and AgeGroup
To analyze relationships between variables and survival outcomes
To visualize insights and trends for further modeling
Key Operations Performed
Duplicate Removal: Ensured dataset integrity by removing duplicates
Missing Value Handling: Filled missing values for Age and Embarked
Outlier Detection: Applied IQR method to detect and handle outliers in Fare
Feature Engineering: Created FamilySize and AgeGroup to improve analysis
Categorical Encoding: Converted Sex, Embarked, and AgeGroup to numerical form
Exploratory Data Analysis (EDA): Visualization of survival trends by gender, class, age, family size, and fare
Correlation Analysis: Identified strongest predictors of survival
Key Insights
Female passengers had higher survival rates than males
First-class passengers survived more than lower-class passengers
Small family sizes correlated with better survival
Passengers with higher fares had better survival chances
Tools Used
Python (Pandas, NumPy, Matplotlib, Seaborn)
Microsoft Excel
